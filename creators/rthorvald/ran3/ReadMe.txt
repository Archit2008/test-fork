This directory contains all files necessary for the Ran III scenario.

There are two packages:
- ran_iii_install1.zip  contains the actual add-on, sans the Hi-Res directory.
- ran_iii_install2.zip contains the Hi-Res directory for the Add-On.

It is split up like so for a more convenient download, as the scenario is very large (3 gigabytes unzipped). Both downloads contains installation instructions.
